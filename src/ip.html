사용 가이드

조회하려는 IP 주소를 입력 후 조회 버튼을 누르거나, 빈칸 상태에서 내 IP 버튼을 클릭하세요. 입력된 IP를 기반으로 위치, ISP, 좌표, 기기 정보를 표시하며 지도로 확인할 수 있습니다.

사례

• IP 비워두고 내 IP 클릭 → 현재 공인 IP 확인 및 위치 파악
• 8.8.8.8 입력 후 조회 → Google DNS 서버 위치 정보 확인

주의사항

  • 위치 정보는 IP 기반 대략적 위치이며, 실제 위치와 차이가 있을 수 있습니다.
  • 개인 IP 정보는 민감 데이터이므로 공개 네트워크에서 주의해 사용하세요.
  • 지도 로드는 외부 API를 사용하므로, 네트워크 상태에 따라 지연될 수 있습니다.

자주 묻는 질문(FAQ)

Q1. 내 IP 조회 시 개인정보는 저장되나요?

클라이언트 측에서만 처리되며 서버나 외부에 저장되지 않습니다. 단, IP 정보는 외부 API로 전송될 수 있습니다.

Q2. 위치 정보가 부정확할 수 있나요?

네. IP 기반 위치는 대략적인 정보만 제공하므로, 실제 위치와 차이가 있을 수 있습니다.
